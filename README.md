Crie um aplicativo de organização de finanças pessoais com as seguintes funcionalidades:

- Cadastro de receitas e despesas
- Classificação automática de gastos usando IA
- Dashboard com gráficos simples (gastos por categoria, saldo mensal)
- Sugestões inteligentes de economia com base no comportamento do usuário
- Alertas de gastos fora do padrão
- Previsão de despesas futuras com base no histórico

Requisitos técnicos:
- Frontend simples e responsivo
- Backend leve (Node.js ou Firebase)
- Banco de dados para armazenar transações
- Integração com IA para análise de dados financeiros

Diferencial:
- O app deve se comportar como um "assistente financeiro pessoal", explicando insights de forma simples e amigável.

Objetivo:
Ajudar usuários iniciantes a organizarem sua vida financeira sem complexidade.
